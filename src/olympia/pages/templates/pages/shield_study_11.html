Plugin Safety

{% trans %} Mozilla wants to figure out the best way to keep our users safe if they choose to use third-party software such as plugins. Participating in this study will help us find out! {% endtrans %}

{% trans %} What will happen next {% endtrans %}

{% trans %} If you agree to participate: {% endtrans %}

  1. {% trans %} We may add security and plugin control features to Firefox to help with online security and performance. {% endtrans %}
  2. {% trans %} Continue using Firefox as you normally would. Depending on how you use the Web, you may not notice any changes. {% endtrans %}
  3. {% trans %} After a few weeks, the study will end on its own. Your Firefox will go back to normal. {% endtrans %}
  4. {% trans %} We will ask you to fill out a quick optional survey to tell us about your experience. The survey should take less than 5 minutes to complete. {% endtrans %}

{% trans %} Leaving the study {% endtrans %}

{% trans %} The study will expire on its own in a few weeks. You may leave the study early. To do so, follow these steps: {% endtrans %}

  1. {% trans %} Type about:addons into the location bar, and press enter. {% endtrans %}
  2. {% trans %} Find Plugin Safety in the addons list. {% endtrans %}
  3. {% trans %} Click the remove button. {% endtrans %}
  4. {% trans %} If you opt out of the study early we will ask you to fill out a survey. We’re interested in hearing about your experience, even if you didn’t participate in the entire study. {% endtrans %}

{% trans %} Opting out of a study does not prevent you from participating in future studies. {% endtrans %}

{% trans %} Your privacy {% endtrans %}

{% trans %} All Shield Studies {% endtrans %}

{% trans %} Every Shield Study collects data about important study events, such as install, uninstall, daily usage, and end of study. Studies also include an optional survey. {% endtrans %}

{% trans url='https://gecko.readthedocs.io/page/toolkit/components/telemetry/telemetry/index.html' %} This data is associated with Firefox Telemetry collection. {% endtrans %}

{% trans %} This particular Study {% endtrans %}

{% trans %} In addition to the data collected by all Shield Studies, Mozilla will collect information about: {% endtrans %}

  • {% trans %} Websites you visit that try to use plugins, and what content on the site requires the plugin; {% endtrans %}
  • {% trans %} How you interact with plugin content on those sites, including whether and how you choose to allow the plugin to run. {% endtrans %}

{% trans %} Mozilla will keep this information private; any data will only be shared in aggregate or anonymized form. {% endtrans %}

{% trans url='https://github.com/squarewave/shield-study-essential-flash/blob/master/docs/metrics.md' %} You can learn more about the data collection for this study here. {% endtrans %}
