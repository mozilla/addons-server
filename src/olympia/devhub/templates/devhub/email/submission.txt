Add-ons help personalize the web experience for millions of users who have installed Firefox, and we want to help make your users happy! If you have not already done so, we recommend checking out these resources:

* For more add-on developer resources, please visit MDN web docs: https://developer.mozilla.org/Add-ons
* Want to ask a question or file a bug report? These are the best ways to contact us: https://developer.mozilla.org/Add-ons#Contact_us
